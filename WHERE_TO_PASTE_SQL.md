### Step-by-Step Instructions:

1. **Open Supabase Dashboard**
   - Go to: https://supabase.com/dashboard/project/vuwkkhmkbtawyqvvqanu

2. **Click "SQL Editor"** (in the left sidebar)
   - It's usually near the top, with an icon that looks like `</>`

3. **Click "New query"** button
   - This opens a blank SQL editor window

4. **Paste your SQL code** into the large text area
   - This is the main editing area where you type/paste SQL
   - It should be a blank white/gray text box

5. **Click "Run"** button (or press Cmd/Ctrl + Enter)
   - The Run button is usually at the bottom right of the SQL editor
